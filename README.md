## 📋 Sobre este proyecto

Este es mi **CV Web profesional** desarrollado inicialmente como parte de una asignatura universitaria. El sitio web presenta mi perfil profesional a través de diferentes formatos y componentes.

### 🎯 Componentes incluidos:
- **Presentación personal** con carta de presentación descargable
- **CV Tradicional** en formato PDF
- **Portfolio visual** (perfil de LinkedIn)
- **Vídeo CV** personal
- **Blog profesional** con artículos técnicos
- **Información de contacto** y redes sociales

## ⚠️ **DISCLAIMER ACADÉMICO**

> **Nota importante:** Esta versión actual contiene algunos elementos específicamente diseñados para cumplir con los **requisitos académicos** de la asignatura. Algunos componentes pueden no reflejar completamente mi experiencia real o preferencias personales, ya que están adaptados a las especificaciones del proyecto universitario.
> 
> **Una vez completada la asignatura**, planeo actualizar y personalizar completamente el contenido para que refleje de manera más auténtica mi trayectoria profesional, proyectos reales y visión personal.

## 🚀 Tecnologías utilizadas

- **HTML5** - Estructura semántica
